# Physio Website

Site statique de démonstration pour le Cabinet Kiné Santé Grenoble.

## Fichiers

- `index.html` : structure de la page
- `style.css` : styles responsive
- `script.js` : menu mobile, FAQ et validation du formulaire
- `assets/cabinet-interieur.jpg` : image hero

## Personnalisation

Remplacer les liens Doctolib `https://www.doctolib.fr/` par le profil réel du cabinet avant mise en ligne.
